# selenium-ecommerce-automation

This project automates the end-to-end checkout flow 
of SauceDemo using Selenium WebDriver and Java.

## Features
- Login automation
- Add to cart
- Checkout flow
- Order validation
- Screenshot capture

## Tech Stack
- Java
- Selenium WebDriver
- ChromeDriver

## How to Run
1. Clone repo
2. Install dependencies and
3. Run test file
